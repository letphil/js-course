## while-loop

```
The while loop is used to execute a block of code as long as the condition is true.

syntax:
while (condition) {
    // code block
}

example:
let x = 0;
while (x < 5) {
    console.log(x);
    x++;
}

other way to write while loop
do {
    // code block
} while (condition);


```
